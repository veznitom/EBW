Detailed Description

Interface functions as bridge between user input and logical structure of a Table.

It manages displaying content of Table and provides it with input data. Interface does not do any extreme calculations and operations. It's main purpose is managing ncurses library and and provide user with UI that is not pure text but has a little style and logic to it.
